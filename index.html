Dúvidas e Atualizações

Nesse site você pode montar sua cartela para o bingo do Discord do canal. Você pode escolher as plantas (arraste para a esquerda para mais opções), ou deixar com que o site escolha para você. Quando terminar, basta clicar no botão de "baixar" para salvar uma foto de sua cartela. Caso queira apagar uma escolha, é só clicar na planta escolhida para remove-la ou clicar no botão de apagar para refaze-la.
